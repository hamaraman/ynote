# Day 2 (화) — 네이버 카페 (월세·전세)

- **채널**: 네이버 카페 (자취/부동산/사회초년생)
- **주제**: 청년 월세 지원 + 청년 전세자금대출(버팀목) + 전세보증금 반환보증
- **형식**: 정보공유 글(하단 출처 링크)
- **소요**: 50분
- **오늘의 원칙**: **가입 즉시 게시 금지.** 등업 조건·활동 이력 채운 뒤 게시. 광고 게시판이 따로 있으면 거기에.

---

## 붙여넣을 콘텐츠 A — 카페 정보글 (월세 지원)

**제목:** `자취러 필독) 청년 월세 지원 놓치는 사람 많아서 정리함`

```
월세 부담 다들 크시죠. 저도 알아보다가 헷갈리는 부분 많길래 정리해봤어요.

[청년 월세 특별지원 핵심]
- 만 19~34세 무주택 청년
- 부모와 실제로 따로 거주(독립세대) + 소득·재산 요건 충족
- 월 최대 20만원 안팎, 최장 12개월 (지자체·공고별 상이)

[신청 전 체크]
1. 복지로 / 거주지 지자체 홈페이지에서 접수기간 확인
2. 임대차계약서 + 월세 계좌이체 내역 미리 준비 (현금납부는 증빙 애매)
3. 국가장학금처럼 소득 심사 있음 → 서류 여유 있게

전세대출(버팀목)이랑 중복되는지는 조건 따라 갈려서 개별 확인해야 해요.

원가구 소득 기준이랑 지역별 차이 정리된 거 참고했어요
출처: https://www.ynote.kr/policy/youth-rent-support?utm_source=naver_cafe&utm_medium=post&utm_campaign=week1

신청해보신 분 후기 있으면 댓글로 공유 부탁드려요!
```

## 붙여넣을 콘텐츠 B — 카페 정보글 (전세보증금 반환보증)

**제목:** `전세 들어가기 전에 '반환보증' 안 들면 진짜 위험합니다 (경험담+정리)`

```
전세사기 뉴스 많아서 다들 걱정 많으시죠. 보증금 지키는 최소 안전장치가
'전세보증금 반환보증'이라 정리해봤어요.

[반환보증이 뭐냐면]
집주인이 보증금 안 돌려줄 때 보증기관(HUG 등)이 대신 돌려주고
나중에 집주인한테 청구하는 구조예요. 내 보증금 안전판이라 보면 됩니다.

[가입 전 꼭 확인]
- 전입신고 + 확정일자는 기본 중의 기본 (이사 당일 바로!)
- 등기부등본에서 근저당·선순위 권리 확인 (내 보증금보다 앞서면 위험)
- 보증료가 들지만 보증금 규모 대비 아주 적은 편이라 보험료라 생각하세요
- 상품·기관마다 가입 가능한 보증금 한도·요건이 달라요

계약 전에 등기부 떼보고, 시세 대비 보증금이 너무 높은 '깡통전세'는 피하는 게
1번입니다. 보증 가입은 그다음 안전장치예요.

가입 요건이랑 청년 전세대출이랑 같이 쓰는 법 정리된 거 봤어요
출처: https://www.ynote.kr/policy/jeonse-loan?utm_source=naver_cafe&utm_medium=post&utm_campaign=week1

곧 전세 계약하시는 분들 안전하게 하시길 바라요.
```

## 보조 콘텐츠 — 지식iN 답변 (청년 전세자금대출, 여력 되면)

**Q. 버팀목 전세자금대출이랑 중소기업 청년 전세대출 뭐가 더 나은가요?**

```
둘 다 청년한테 좋은 상품인데 결이 달라서 상황별로 갈려요.

[중소기업 취업청년 전세대출]
- 중소·중견기업 재직자 또는 청년창업자 대상
- 금리가 아주 낮은 편(연 1%대 수준)이 최대 강점
- 대신 '중소기업 재직' 요건이 핵심이라 조건이 좁음

[버팀목 전세자금대출]
- 무주택 청년 폭넓게 대상 (재직요건 없음)
- 금리는 중기청보다는 높지만 여전히 시중보다 저렴
- 소득·보증금·전용면적 한도 요건 충족해야 함

정리하면, 중소기업 다니면서 요건 맞으면 → 중기청이 금리로 유리하고,
재직요건이 안 맞거나 프리랜서/공공 등이면 → 버팀목이 현실적이에요.

공통 팁: 대출 실행 전에 집이 근저당·권리관계 깨끗한지 꼭 확인하세요.
승인 나도 물건 하자로 막히는 경우가 있어요.

한도·금리·자격 요건 비교표는 여기 봤어요
→ https://www.ynote.kr/policy/jeonse-loan?utm_source=naver_kin&utm_medium=answer&utm_campaign=week1
```
