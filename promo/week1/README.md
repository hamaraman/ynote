## KPI (Day 7 결산 기준)

측정: **GA4(G-K3CGSNJ5VK)** + Google Search Console

| 지표 | 어디서 | 1주 현실적 목표 | 판단 |
|---|---|---|---|
| 총 방문자 | GA4 > 사용자 획득 | 50~150명 | 0→두 자릿수면 채널 작동 중 |
| 채널별 세션 | GA4 > 트래픽 획득(utm_source별) | 편중 확인 | 어느 채널이 클릭을 주는지 |
| 검색 노출/클릭 | Search Console > 실적 | 노출 수백 단위 | 색인·검색 반응 시작 여부 |
| 참여 시간 | GA4 참여도 | 평균 30초↑ | 콘텐츠가 기대와 맞는지 |
| 인스타 저장 | 인스타 인사이트 | 저장 10+ | 카드뉴스 확산성 |

**다음 주 조정 규칙**
- utm 세션 1위 채널 → 배포량 2배 집중
- 방문 있는데 참여 30초 미만 → 제목이 낚시, 랜딩 보강
- 에타·디시에서 삭제/신고 → 링크 없이 '언급'만으로 전환
- 지식iN 채택/조회 상승 → 답변 개수 주 10건으로 확대

- ❌ 디시/클리앙 초보 링크 (텃세 역효과, 이번 주는 눈팅만)
