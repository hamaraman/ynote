# Day 5 (금) — 지식iN + 오픈채팅 (주거)

- **채널**: 네이버 지식iN + 카카오톡 오픈채팅
- **주제**: 행복주택 / 청년 매입임대주택
- **형식**: 지식iN 답변 2개 + 오픈채팅 자연 답변
- **소요**: 40분
- **오늘의 원칙**: 오픈채팅은 **입장 직후 홍보 금지(강퇴 1순위)**. 대화 몇 마디 후 질문이 나올 때 답으로 링크.

---

## 붙여넣을 콘텐츠 A — 지식iN 답변 (행복주택)

**Q. 행복주택 청년이면 아무나 신청 가능한가요? 소득 조건 있나요?**

```
행복주택은 '아무나'는 아니고 계층별 자격이 있어요. 청년 계층 기준으로 알려드릴게요.

■ 대상: 만 19~39세 청년, 대학생, (사회초년생 포함) 무주택자
■ 소득: 본인/가구 소득이 도시근로자 월평균소득 기준 이하 (모집공고마다 %기준 명시)
■ 자산: 총자산·자동차가액 기준도 있음
■ 임대료: 주변 시세의 60~80% 수준 (역세권 신축이라 인기 높음)

거주기간은 보통 최대 6년(대학생·청년), 조건 충족 시 연장되는 유형도 있어요.

핵심은 '공고 단위'로 뜬다는 거예요. LH·SH청약센터에서 모집공고가 떠야
신청 가능하고, 지역·단지마다 자격·소득기준·경쟁률이 다 달라요.
관심 지역 알림 설정해두고 공고 뜨면 바로 서류 준비하는 게 현실적입니다.

소득·자산 기준이랑 청년 유형별 자격은 여기 정리돼 있어서 참고했어요
→ https://www.ynote.kr/policy/happy-house?utm_source=naver_kin&utm_medium=answer&utm_campaign=week1
```

---

## 붙여넣을 콘텐츠 B — 지식iN 답변 (청년 매입임대주택)

**Q. 청년 매입임대주택이 행복주택이랑 다른 건가요? 임대료 얼마예요?**

```
비슷해 보이지만 방식이 달라요. 간단히 구분해드릴게요.

매입임대는 LH 등이 기존 주택을 사들여 청년에게 저렴하게 재임대하는 방식이에요.
(행복주택은 대개 신축 공공임대라는 차이가 있어요.)

■ 대상: 만 19~39세 무주택 청년 (대학생·취준생 포함 유형 있음)
■ 임대료: 주변 시세보다 크게 저렴한 수준 (시세의 40~50%대인 유형도 있음)
■ 보증금: 유형에 따라 낮게 책정, 거주기간도 정해져 있음(연장 가능 유형 有)
■ 소득·자산: 계층·순위별 소득/자산 요건 충족해야 함

가장 큰 장점은 임대료 부담이 확 낮다는 거고,
단점은 '기존 주택'이라 위치·연식·컨디션이 물건마다 편차가 크다는 점이에요.
그래서 공고 뜨면 입지랑 상태를 잘 보고 지원하는 게 중요해요.

LH청약센터에서 지역별 모집공고로 뜨니까 관심지역 알림 걸어두세요.

순위별 자격이랑 임대료·거주기간 정리된 건 여기 참고했어요
→ https://www.ynote.kr/policy/youth-purchase-rental?utm_source=naver_kin&utm_medium=answer&utm_campaign=week1
```

## 오픈채팅 운영 팁
- '자취', '취준', '청년정책', '부동산' 키워드로 방 검색 후 입장
- 최소 몇 마디 대화 후, 주거 관련 질문이 나오면 그때 답으로 링크
- 예: "행복주택이랑 매입임대 차이 궁금하면 이거 정리 잘 돼있어요 → (링크)"
