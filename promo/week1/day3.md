## 카드 5장 문구 (기후동행카드)

```
[카드 1]
서울 사는 청년,
아직도 교통비 다 내세요? 🚇
월 6만원대로 무제한 타는 법

[카드 2]
기후동행카드란?
지하철 + 버스 + 따릉이까지
서울권 대중교통 한 달 무제한권

[카드 3]
가격은?
✔ 월 6만원대
✔ 만 19~34세 청년은 추가 할인
✔ 따릉이 포함 / 미포함 선택 가능

[카드 4]
이런 사람은 무조건 이득 💸
하루 왕복 3천원 이상 &
주 5일 이상 출근·통학하는 사람
→ 한 달 교통비 계산해보면 답 나옴

[카드 5]
신청 방법은?
편의점(실물) or 앱(모바일)에서 발급·충전
청년 할인은 본인확인 절차 필요
👉 자세한 건 프로필 링크 확인
```

## 캡션
```
서울 대중교통 한 달 무제한, 기후동행카드 총정리 🚇

버스 + 지하철 + 따릉이까지 월 6만원대.
특히 만 19~34세 청년은 추가 할인까지 적용돼요.

한 달 교통비 7~8만원 넘게 쓴다면? 무조건 이득 💸
정기권 끊기 전에 본인 통근 패턴이랑 비교 필수!

신청·청년 할인 조건은 프로필 링크에서 👆
저장해두고 필요할 때 꺼내보세요 📌
```

## 보조 콘텐츠 — 지식iN 답변 (기후동행카드, 여력 되면)

**Q. 기후동행카드 청년 할인 따로 있나요? 6만원대 그거 맞아요?**

```
맞아요, 청년은 추가 할인이 적용돼서 더 저렴하게 쓸 수 있어요.

■ 기본: 서울 지하철 + 시내버스 + 따릉이까지 월 무제한, 6만원대
■ 청년(만 19~34세): 추가 할인 적용가로 이용 가능 (본인 인증 필요)
■ 종류: 따릉이 포함형 / 미포함형이 있어서 가격이 조금 달라요

주의할 점 몇 가지 정리하면요.
- 기본은 '서울 지역' 대중교통 위주예요. 신분당선, 광역버스 등 일부 노선은 제외될 수 있어서 본인 출퇴근 경로 확인 필수.
- 실물카드는 편의점 구매 후 충전, 모바일은 앱(안드로이드 중심)으로 발급·충전해요.
- 청년 할인은 별도 본인확인 절차를 거쳐야 적용됩니다.

한 달 교통비가 대략 7~8만원 넘게 나오면 거의 무조건 이득이에요.
정기권 끊기 전에 본인 통근 패턴이랑 한번 비교해보세요.

청년 할인 신청 방법이랑 적용/제외 노선은 여기 정리돼 있어요
→ https://www.ynote.kr/policy/climate-card?utm_source=naver_kin&utm_medium=answer&utm_campaign=week1
```
